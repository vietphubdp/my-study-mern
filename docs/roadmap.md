# Roadmap đề xuất (12 tuần)

## Sprint 1-2
- Setup monorepo hoặc tách frontend/backend.
- Auth register/login/logout/refresh.
- RBAC middleware + policy checks.
- User profile (private by default).

## Sprint 3-4
- CRUD Course/Topic/Lesson/Note.
- Markdown editor + autosave cơ bản.
- Tagging + filter local.
- Upload ảnh/attachments qua Cloudinary.

## Sprint 5-6
- Flashcard CRUD + thuật toán Leitner.
- Màn hình review thẻ đến hạn.
- Quiz engine cơ bản + timer.
- Lưu attempt + thống kê accuracy.

## Sprint 7-8
- Dashboard: streak, due cards, study time.
- Search server-side + index Mongo.
- Export/Import CSV/JSON.
- Hardening auth/security/logging.

## Sprint 9-10 (V1.5)
- Cải thiện UX, pagination, empty states.
- Email verify + forgot/reset password hoàn chỉnh.
- Tối ưu truy vấn analytics.

## Sprint 11-12 (V2 social)
- Share public/unlisted/private.
- Fork/copy content.
- Report + moderation workflow (admin).
- Heatmap học tập.
